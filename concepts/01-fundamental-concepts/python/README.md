# Fundamental Concepts - Python Examples

This directory contains Python implementations of fundamental system design concepts.

## Files

- `load_balancer.py` - Load balancing algorithms (Round Robin, Least Connections, IP Hash, Weighted Round Robin)
- `cap_theorem.py` - CAP theorem demonstrations (CP, AP, CA systems)
- `acid_transaction.py` - ACID transaction implementation with money transfer example

## Running Examples

```bash
# Load Balancer
python load_balancer.py

# CAP Theorem
python cap_theorem.py

# ACID Transactions
python acid_transaction.py
```
